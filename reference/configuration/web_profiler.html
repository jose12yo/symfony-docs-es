Configurando WebProfiler¶

Configuración predeterminada completa¶

  • YAML
    web_profiler:

    # mostrar información secundaria para acortar la barra de herramientas
    verbose:             true

    # mostrar la barra de depuración web en la parte inferior de
    # las páginas con un resumen de información del perfil
    toolbar:             false

    # te da la oportunidad ver los datos recogidos antes de seguir la redirección
    intercept_redirects: false
